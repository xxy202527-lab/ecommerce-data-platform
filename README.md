### 🚀 快速开始  
```bash
# 克隆仓库
git clone git@github.com:xxy202527-lab/ecommerce-data-platform.git
cd ecommerce-data-platform

# 安装依赖
pip install -r requirements.txt

# 配置数据库（见 config/db_config.md）
```
